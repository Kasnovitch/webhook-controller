# Dctools

**Dctools** is a C# console application for interacting with Discord webhooks and guilds. It provides various utilities such as sending messages, managing webhooks, viewing guild information, and more. Designed with a colorful console interface and playful “poison flavor” animations.

---

## Features

- Send messages to Discord webhooks
- View guild information
- List members of a guild
- Delete webhooks
- Rename webhooks
- Spam messages to a webhook
- Change webhook avatar
- Credits page
- Colorful console UI with animated effects

---

## Requirements

- Windows 10 or later
- [.NET Framework 4.7.2](https://dotnet.microsoft.com/download/dotnet-framework/net472) or newer
- Internet connection for webhook requests

> ⚠️ **Important:** Use responsibly. Avoid spamming webhooks on servers you do not own. This tool is intended for testing and personal use only.
